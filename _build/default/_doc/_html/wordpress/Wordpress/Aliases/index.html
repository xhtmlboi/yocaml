Module Wordpress.Aliases

Some type aliases to clarify the signatures of some functions.

type filepath = string

At the moment, I don't have an efficient abstraction to describe a file path. It's a bit sad but that's life...

type directory = string

At the moment, I don't have an efficient abstraction to describe a directory path. It's a bit sad but that's life...

type e

e is a type that only serves to mark the ghost type of the effects definition. This is a bit shorter than writing unit.
